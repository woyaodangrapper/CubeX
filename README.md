## **安装下载**

> #### 在 [tags](https://github.com/Xcube-Studio/Natsurainko.FluentLauncher/tags) 下载启动后点击设置对启动器进行初步的配置 如游戏目录的选取 Java 运行时的添加 游戏帐户的添加

### 运行前的配置要求

> Windows 10.0.19041.0 及以上系统版本。
